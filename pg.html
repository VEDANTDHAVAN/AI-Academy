Functions

In Data Structures and Algorithms (DSA), functions are blocks of code designed to perform specific tasks, such as data manipulation, searching, sorting, and problem-solving, allowing modular and reusable code.

Stack

A stack is a linear data structure that follows the Last In, First Out (LIFO) principle, where elements are added (pushed) and removed (popped) from the top, making it useful for tasks like function calls, undo operations, and expression evaluation.
